# Cover images

Drop cover images for wirds and collections here. They are referenced by path
from `wirds.json` (`cover` field). v1 doesn't actually render these inside the
app cards yet — they're plumbed through the data layer so the next version can
turn them on without a model change.

Recommended size: 1080×720 JPG or PNG.

This README exists to keep the directory present in version control. The
`flutter pub get` step requires the asset directory to exist when listed in
`pubspec.yaml`.
